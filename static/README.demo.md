# Демо ИИ-ассистента для дисциплин web-интерфейсов

**Команда:** Трубчик · Ревякина · Гнедина · кафедра ИСТ · ДГТУ
**Дата защиты:** 9 июня 2026 г.

## Что это

Standalone-демо, упакованное в один HTML + один JS-файл. Запускается без сервера, npm и Docker — двойным кликом на `index.html` в любом современном браузере. Содержит:

- 3 рабочих кейса из презентации (зоны A, C, D)
- Живой редактор кода с эвристическим линтером HTML / CSS / JS
- Карточки базы знаний с 8 методическими темами
- Телеметрию и метрики, бейджи зон делегирования

## Как показывать на защите

1. Откройте `index.html` в полноэкранном режиме браузера (F11).
2. В правой панели — 4 готовых кнопки сценариев:
   - **A · Что такое flexbox?** — типовой вопрос, ассистент отвечает с ссылками на методички.
   - **A · Зачем нужны семантические теги HTML5?** — резервный вопрос той же зоны.
   - **C · Объясни Composition API в Vue 3** — эскалация к преподавателю, явный отказ от объяснения новой темы.
   - **D · Поставь мне оценку** — заблокировано архитектурно с пояснением трёх красных линий.
3. Переключитесь на вкладку **Редактор + превью** → нажмите **Проверить код**.
4. Покажите вкладку **База знаний** — корпус методичек и телеметрию.

## Структура файлов

```
index.html     - вёрстка, стили, разметка (один файл)
app.js         - вся логика: сценарии, линтер, рендеринг
README.md      - этот документ
```
